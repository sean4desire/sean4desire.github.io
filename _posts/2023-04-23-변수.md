## 변수

하나의 값을 저장할 수 있는 메모리 공간

### 변수의 선언방법

`변수타입 변수이름;`

메모리의 빈 공간에 변수타입에 알맞은 크기의 저장공간이 확보되고, 이 저장공간은 변수이름을 통해 사용할 수 있게 된다.

### 변수의 초기화

`대입연산자 =`

적절한 값을 변수에 저장하여 쓰레기값을 덮어써야 한다.

## 예제 1

```C
#include <stdio.h>

  

int main(void) {

int age = 30;

int year;

  

printf("year: %d\n", year);

year = age + 2000;

  

printf("age: %d\n", age);

printf("year: %d\n", year);

  

age = age + 1;

printf("age: %d year: %d\n", age, year);

  

return 0;

}
```
```bash
(실행결과)
year: 32759

age: 30

year: 2030

age: 31 year: 2030
```

## 예제 2

```C
#include <stdio.h>

  

int main(void) {

int x = 10;

int y = 20;

int tmp;

  

printf("x: %d y: %d\n", x, y);

  

tmp = x;

x = y;

y = tmp;

  

printf("x: %d y: %d\n", x, y);

return 0;

}
```
```bash
(실행결과)
x: 10 y: 20

x: 20 y: 10
```
