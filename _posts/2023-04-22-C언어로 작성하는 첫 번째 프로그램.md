## 프로그램

컴퓨터가 어떤 작업을 수행하는데 필요한 명령들을 (순서대로) 작성한 것

## 컴파일

인간어를 기계어로 번역하는 일

c 확장자 파일이 obj 확장자 파일로 변환된다.

## 링크

obj 확장자 파일들을 합쳐 하나의 실행(exe)파일을 만드는 일

## 빌드

컴파일 + 링크

## C언어로 작성하는 첫 번째 프로그램

```C

#include <stdio.h> // printf()에 대한 정보를 컴파일러에게 제공

int main(void)
{
	printf("Hello, world.");
	
	return 0;
}
```
